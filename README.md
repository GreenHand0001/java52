# 中医药材后台管理系统 java52

## 项目概述

中医药材后台管理系统是一个基于前后端分离架构的信息管理平台。该系统致力于提供中药材的信息管理、库存监控、数据分析和药材知识库等功能，适用于需要对中药材进行全方位管理的场景。

## 技术栈

- **前端**：Vue3、Element Plus、Echarts
- **后端**：Java、SpringBoot
- **数据库**：Mysql

## 系统角色

- 管理员

## 运行环境

- **系统环境**：Windows，Mac
- **开发环境**：IDEA、JDK1.8、Mysql

## 功能模块

### 管理员功能

- **首页管理**：
  - 仪表盘统计（药材总数、分类数量、订单数量、用户总数等）
  - 数据可视化图表（订单统计、热门药材）

- **药材管理**：
  - 药材列表、添加药材、修改药材、删除药材
  - 药材分类管理、药材状态管理、库存管理、图片上传

- **订单管理**：

- **用户管理**：
  - 用户列表、用户状态管理（启用/禁用）、用户删除

## 核心亮点

- **权限控制**：基于SpringSecurity + JWT的安全认证
- **界面设计**：基于Vue3 + Element Plus的响应式设计
- **数据统计分析**：完善的数据可视化展示
- **代码规范**：采用MVC架构，代码结构清晰
